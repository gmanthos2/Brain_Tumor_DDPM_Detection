\begin{abstract}
We present a latent diffusion model for brain MRI anomaly detection and synthetic image generation. A Variational Autoencoder (VAE) compresses 256$\times$256 grayscale brain MRI slices into a compact latent space, where a Denoising Diffusion Probabilistic Model (DDPM) learns the distribution of healthy brain anatomy using Classifier-Free Guidance. For anomaly detection, input images are partially noised and reconstructed conditioned on the healthy class; reconstruction errors localize pathological regions such as tumors. For data augmentation, the model generates high-quality synthetic brain MRIs by sampling from the learned distribution. We evaluate anomaly detection using AUROC and AUPRC, and generation quality using FID, SSIM, and LPIPS. Additionally, we provide explainability through attention map visualization and denoising trajectory analysis, and discuss ethical considerations for clinical deployment.
\end{abstract}
